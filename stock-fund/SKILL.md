# 用法

```bash
# 按代码查询估值
node <本 SKILL.md 所在目录>/scripts/query.mjs <基金代码> [基金代码2 ...]

# 按名称搜索基金代码
node <本 SKILL.md 所在目录>/scripts/query.mjs search <关键词>
```

## 流程

1. 用户提供基金代码 → 直接查询估值
2. 用户提供基金名称但不确定 → 先 `search` 搜索，列出候选让用户选择，再根据选定代码查询
3. 用户提供精确名称且搜索结果唯一 → 直接查询该基金

# 数据来源

天天基金估值接口：`http://fundgz.1234567.com.cn/js/{基金代码}.js`

返回字段：

| 字段 | 含义 |
|------|------|
| `fundcode` | 基金代码 |
| `name` | 基金名称 |
| `dwjz` | 单位净值（上一交易日收盘） |
| `gsz` | 估算净值（盘中实时） |
| `gszzl` | 估算涨跌幅（%） |
| `jzrq` | 净值日期 |
| `gztime` | 估值时间 |

## 注意事项

- 估算净值为盘中估算，官方净值于收盘后公布
- 非交易时间查询显示的是上一交易日的数据
- A 股惯例：🔴 红色表示上涨，🟢 绿色表示下跌
- **开放式基金**（场外）不能实时行情，只有估算净值（盘中）和官方净值（收盘后 T+1）
